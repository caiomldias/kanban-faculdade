# Kanban da Faculdade

Projeto acadêmico desenvolvido para organizar tarefas da faculdade no dia a dia, com foco em priorização, acompanhamento de progresso e melhoria da rotina de estudos.

## Proposta

O **Kanban da Faculdade** é uma aplicação web simples para transformar tarefas acadêmicas em um quadro visual de acompanhamento. A ideia é separar o que precisa ser feito, o que está em andamento, o que já foi concluído e quais impedimentos estão atrapalhando a evolução.

O projeto também conecta organização pessoal com desenvolvimento profissional, registrando uma aba específica de **Desenvolvimento Pessoal FIAP** para acompanhar evolução acadêmica, prática em projetos e construção de portfólio.

## Problema trabalhado

Durante a rotina da faculdade, várias tarefas competem pela atenção: trabalhos, provas, leituras, atividades avaliativas, mensagens de grupo e prazos diferentes. Sem uma visão clara, fica mais difícil decidir o que é realmente importante e o que pode esperar.

O projeto parte desse problema para criar uma interface objetiva, inspirada em Kanban, que ajude a visualizar prioridades e reduzir a sensação de desorganização.

## Solução proposta

- Quadro Kanban com colunas **To Do**, **Doing** e **Done**
- Cadastro de tarefas com matéria, importância e prazo
- Separação visual entre tarefas importantes e não importantes
- Área de impedimentos para registrar bloqueios
- Checklist de Definition of Done para revisar tarefas antes de concluir
- Aba de Desenvolvimento Pessoal FIAP para acompanhar progresso de carreira
- Layout responsivo para diferentes tamanhos de tela
- Salvamento local no navegador usando `localStorage`

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Tailwind CSS via CDN
- Git e GitHub

## Como abrir

Abra o arquivo `index.html` no navegador ou rode um servidor local:

```bash
python3 -m http.server 4173
```

Depois acesse:

```text
http://localhost:4173
```
